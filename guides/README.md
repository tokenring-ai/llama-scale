# Guides

Platform-specific getting started guides for [llama-scale](https://github.com/tokenring-ai/llama-scale).

| Guide | Best for |
|-------|----------|
| [Debian / Ubuntu](getting-started-debian-ubuntu.md) | `.deb` package + systemd |
| [Red Hat / Fedora](getting-started-redhat-fedora.md) | `.rpm` package + systemd |
| [macOS](getting-started-macos.md) | Homebrew-style local install (binary, npm, or Cargo) |
| [Docker](getting-started-docker.md) | Containers and GHCR images |
| [Kubernetes](getting-started-kubernetes.md) | Deployment, Service, probes, Ingress |
| [Proxmox VE](getting-started-proxmox.md) | LXC package install, Docker guest, LAN backends |
| [MikroTik](getting-started-mikrotik.md) | RouterOS containers (veth, mounts, NAT) |

For full configuration options, routing behavior, and metrics, see the
[main README](../README.md).
